# 第二章
## 第一节 界面介绍
  1. 工作区右键倒三角选择列中可以添加想要的属性
  2. 布局可以拖动更改，在主页-布局选项里有“默认”选项可一键默认
  3. 命令历史记录可以看见上次关闭前的命令
     - 可以在历史记录里直接“复制粘贴”  
     - ==注意：数字键的“上下箭头”是翻看上次/下次命令的快捷键==
## 第二节 脚本
  1. 创建脚本（,m）
     - ctrl ＋ n 快速创建
   - 在主页新建脚本
  2. 输入代码
     - shift + 左键，选择到鼠标所在位置的代码行
  3. 保存代码
     - ctrl + s 快速保存
     - 建议用记事本打开（ ctrl + 滚轮放大/缩小进行查看）
  4. 执行代码
     - 点击“运行”
     - ctrl + a 全选代码，右键—执行所选内容
     - 左键可选部分代码进行运行—“F9”（前提是清空工作区）
     - 直接在命令行窗口输入文件名（不加后缀）可以直接运行
     - 运行代码的文件必须在当前文件夹内（也就是左侧栏）
## 第三节 实时脚本
  1. 创建实时脚本（.mlx）
     - 类似于markdown
     - 在主页直接新建实时脚本
  2. 重要快捷键 
     - ==★★★★★：ctrl + E 转换文本和代码==  /  ==也可以直接创建代码的空白文本行==（代码是有灰色背景的）
     - 内嵌/右侧：指代码运行结果的显示位置
     - 分节符：ctrl + alt + enter
     - ctrl + 数字键的上下键  来切换小节
     - ctrl + 滚轮 切换实时编辑器的页面大小
     - ==每个代码行后面 + “；”分号可以“隐藏代码”-即在运行时不单独导出==
  3. 高版本功能（控件/任务）
     - 数据滑块—即“变阻器”进行变值，导出结果也是实时更新
  4. 导出/保存
## 第四节 文件管理
  1. 文件执行
     - 当执行MATLAB某个文 件的命令时， MATLAB首先会在==当前文件夹==中寻找该命令对应的文件，如果找到的话就会执行这个文件；如果没有找到的话，会在==搜索路径==中继续寻找，如果都没找到的话则会报错。
     - 如果执行的文件名（变量/函数）在工作区出现，则执行顺序改为==工作区→当前文件夹→搜索路径。
     - 工作区文件保存格式为.mat（无需字母开头）
  2. 修改文件夹
     - 右键单击编辑器打开的.m文件名称，选择第一个选项
     - 点击浏览文件夹，然后选择需要修改到的文件夹即可
     - 在命令行窗口执行命令： cd 命令后面接上要修改的文件夹路径:\\  cd 文件路径  /  cd('文件路径')
## 第五节 帮助系统
  1. 在MATLAB 官网搜索
     - 打开MATLAB的帮助中心：https://ww2.mathworks.cn/help/index.html
  2. 使用doc 命令
     - 在命令行窗口输入doc即可调出MATLAB的帮助中心。doc后面可以跟上你要查询的命令名称（doc+空格+命令）
  3. 使用help命令
     - 在 MATLAB的命令行窗口输入help 加上你要查询的命令名称，就会在窗口内返回给我们这个命令的帮助信息
  4. 使用edit命令
     - edit 是编辑的意思，如果直接在命令行窗口输入edit, 就会帮我们创建一个新的脚本文件，并自动打开编辑器。
     - 开头加“%”代表注释的意思，不影响运行结果
     - MATLAB还专门设计了为选中行添加注释、取消注释、注释换行的快捷键，分别是Ctrl+R、Ctrl+T、Ctrl+J
##  第六节 MATLAB的变量
  1. 在 MATLAB中变量的命名应遵循如下规则
     - 变量名必须以字母开头，之后可以是任意的字母、数字或下划线_。
     - 变量名区分字母的大小写，例如a 和 A 代表不同的变量。
     - 变量名不超过63个字符，第63个字符以后的字符将被忽略。
     - 不能定义与MATLAB关键字同名的变量(例如 if 或 end)。要获取关键字的完整列表，请在命令行输入 iskeyword 并运行。
  2. MATLAB使用等号“=”给变量赋值
     - ==开头“%%＋空格”代表着注释的分节（查看/运行和实时脚本一致）==
     - 特殊变量
       | 特殊变量 | 描述 |
       | :----| :---- |
       | ans | 系统默认的用于保存运算结果的变量名 |
       | pi | 圆周率π |
       | inf/-inf | 无穷大和负无穷大，注意1/0=inf |
       | NaN | 不定值或缺失值。例如计算0/0或0*Inf会返回NaN |
       | i和j | 复数中的虚数单位，例如3+4i和3+4j表示同一个复数 |
       | eps | 浮点相对精度。这里涉及到浮点数计算精度的问题，学过C语言的同学应  该知道，计算机内部使用二进制01对浮点数进行编码，如果小数点后面位 数过长，计算机就无法准确表示。这里你可以简单的将eps看成一个非常  小的数，这个数大约等于2.22×10-16,有时候不想分母为0时，我们就可  以在分母上加上eps。(未来涉及浮点数的关系运算时，我们还会谈到这一 点 ) |
  3. 数值更改
     - 临时更改
     - 在运行代码前，我们在命令行窗口输入 format long g /  format longG
     - 永久更改
     - 我们在MATLAB 的设置中永久更改。大家点击 MATLAB 菜单栏——主页——预设，就 能打开MATLAB  的设置界面。我们可以点击命令行窗口，将数值格式设置为long g即可
  4. 小技巧
     - 使用clc 命令可以清空命令行窗口；使用clear 命令可以清空工作区
     - 一些计算结果并不需要在命令行显示，那么我们可以在代码行的最后加上英文分 号“;”分号不能去掉，它起着隔开两行（用一行两个相连）命令的作用
## 第七节 数学运算函数
  1. 运算
     - 可以通过添加小括号的方式改变运算的先后顺序
  2. ==易错点==
     - MATLAB的乘号不能省略。例如计算sin(2π)时，我们要在MATLAB  中输入sin(2*pi), 中间的乘号不能省略。
     - 在对变量命名时，我们不要将变量命名为函数的名称，否则会导致调用函数时出错。
  3. 常见函数
     | 特殊变量 | 描述 |
       | :----| :---- |
       | abs | 求绝对值，也可以用来计算复数 的模长 |
       | floor | 朝着负无穷大方向进行取整 |
       | fix | 朝着零方向进行取整 |
       | ceil | 朝着正无穷大方向进行取整 |
       | round | 真正意义上的四舍五入函数。 |
       | round(X,N) | 可以将X在第N位数四舍五入，分下面三种情况：<br>1.N>0:四舍五入到小数点右侧的第N位数。<br>2.N=0:四舍五入到最接近的整数。<br>3.N<0:四舍五入到小数点左侧的第N位数 。 |
       | mod | mod(a,m)可以计算a除以m后的余数，其中a是被除数，m是除数。(被除数÷除数=商……余数) |
       | sqrt | sqrt(a)可以计算a的平方根，即对 a开根号。如果a为负数则返回复数结果。其结果和a^(1/2)等价。 |
       | exp | exp函数可以计算以自然常数e为 底的指数。 |
       | log | log(x)用来计算以自然常数e为底 数的对数。 |
       | log2/log10 | 分别用来计算以2和10为底的对 数。(注意：MATLAB没有定义log3、log4这种函数!) |
