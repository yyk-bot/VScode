# <font faze="宋体" color=red>一级标题</font>
  - font代表设置字体，后面跟faze / color / size
  - 需要注意的是前后的格式`<font>   </font>`
  - "  /  "表示结束
## <center><font faze="粗体" color=blue size=5>二级标题</font></center>
  - center代表居中
  - 需要注意的是前后的格式`<center>   </center>`
### 三级标题
### 一、准备工作
1. **安装vscode
2. 下载插件
3. 创建.md文件
### 二、基本语法
1. **标题**
   #一级
   ##二级
   ...
2. **引用**
   >内容
   >>内容2
3. **列表**
   1. 无序列表
   - 列表1
   + 列表2
   * 列表3
   2. 有序列表
   3. 嵌套
   4. To do list
      - [ ] a
      - [ ] b
      - [ ] c
4. **表格** # 不用在乎横线数目
   | 左对齐 | 居中对齐 | 右对齐 |
   | :--- | :-----: | ----: |
   | a | bb | ccc |
5. **段落**
   - 换行————两个以上的空格加回车 #  现在没啥必要
   - 分割线————三个及以上的*   # *空格前的数目表示起始位置 
  ***  
   - 字体  # ``展示代码
     | 字体 | 代码 |
     | :---: | :---: |
     | *斜体* | * * |
     | ==高亮== | == == |
     | **粗体** | ** ** |
     | ***斜粗体*** | *** *** |
     | ~~删除~~ | ~~ ~~ |
     | <u>下划线</u> | `<u> </u>` |
   - 脚注 # 必须要解释才能将脚注渲染出来
     引用文献[^1]进行论证！
6. **代码**  # 引用多行“  ```  ”，引用单行“  `  ”
   ```
   print(a)
   ```
   `print(a)`
7. **超链接**  # 注释与正文之间要空行
   - 【如有需要请点击链接】:https://www.bilibili.com/video/BV1bK4y1i7BY?vd_source=cf6a59feffb252d4d3dc2073f77f4341
   - 查看更多功能请[点击链接][教程]
8. **图片**
   - 建议看视频
   - 需要图床
### 其他操作 
   - “  \  ”表示同一行中间空一格，“  `\\`  ”表示换行
   - 输入latex公式  # 用$符号，类似代码引用（单个用一个$，多行用两个$$）
   - 单行
   $f(x)=ax+b$
   - 多行
   $$
   f(x)=ax+b\\
   g(x)=cx+d
   $$
   - **html和css用法**
     - 用于图片的粘贴（详细看视频）
     - ctrl+shift+p 搜索“MPE:CSS”
     - 在style使用css语言更改格式，例如：
     ```
      h1,h3{
        text-align: center;
      }  
     表示标题居中，h1,h3代表1、3级标题
     ```
     - **个性化设置**
       File preferences Settings
### 四、导出为PDF
   - 使用markdown PDF（右键export pdf）（不推荐，有很多格式显示不出）
   - open in browser(手动保存为PDF文档)
